How To

Welcome to SuperFrets! This is a dynamic fretboard tool to help guitar players visualize chord changes.

Add a chord/scale to the right and use your number keys to cycle through the progression ('0' will display chord 10). Clicking on a chord will remove it from the progression.

This tool is still in its infancy, so stay tuned for more updates!
